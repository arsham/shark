*core_tips*


+---------------+---------------+-------------+----------------------------------------------------------------------------------------+
| where         | command       | alternative | notes                                                                                  |
+===============+===============+=============+========================================================================================+
| Terminal Mode | ctrl-w N      |             | to enter terminal-normal mode. You can move around and yank stuff. Type "i" to return. |
| Terminal Mode | ctrl-w r      |             | open command mode                                                                      |
| Terminal Mode | ctrl-w ""     |             | paste from " register                                                                  |
+---------------+---------------+-------------+----------------------------------------------------------------------------------------+
| Command Mode  | :read !ls -l  |             | print the output into the buffer                                                       |
| Command Mode  | :.!figurine   |             | run the command with the line as its stdin, and put the output back in the buffer      |
| Command Mode  | :.!sh         | :.!zsh      | run the current line as a shell command                                                |
+---------------+---------------+-------------+----------------------------------------------------------------------------------------+
| Terminal      | grep 666 -l * |             | xargs -o vim     " -l is for only file names, -o is to give back the stdin to vim      |
+---------------+---------------+-------------+----------------------------------------------------------------------------------------+
